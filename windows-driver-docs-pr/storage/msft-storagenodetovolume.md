The following syntax is simplified from Managed Object Format (MOF) code.

## Syntax

``` syntax
[Association]
class MSFT_StorageNodeToVolume
{
  MSFT_StorageNode REF StorageNode;
  MSFT_Volume      REF Volume;
};
```
