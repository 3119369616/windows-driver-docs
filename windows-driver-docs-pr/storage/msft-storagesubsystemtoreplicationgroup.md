# MSFT\_StorageSubSystemToReplicationGroup class

## Syntax

``` syntax
[Association]
class MSFT_StorageSubSystemToReplicationGroup
{
  MSFT_StorageSubSystem REF StorageSubSystem;
  MSFT_ReplicationGroup REF ReplicationGroup;
};
```
